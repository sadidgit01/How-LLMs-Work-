01Stage

The Prompt

Everything begins with text. A user types a question, a system prompt is prepended, special tokens may frame the conversation. The model receives a single string.

Key insight

The model has no memory. Only what's in the window exists.

[SYSTEM] You are a helpful assistant.
[USER] The meaning of life is
02Stage

Tokenization

Text is split into tokens — chunks that are often words, sometimes sub-words. Each token maps to an integer ID.

Key insight

Tokens ≠ words. A model never sees 'words' — only integer IDs.

03Stage

Embedding

Each integer ID becomes a vector — a list of typically 768 to 12,288 floating-point numbers. This is the model's learned representation.

Key insight

Meaning is geometry. Semantics live in 12,288-dimensional space.

04Stage

Positional Encoding

Transformers process all tokens simultaneously. We add a position vector to tell the model 'this is token number three'.

Key insight

Without position info, 'dog bites man' equals 'man bites dog'.

05Stage

Self-Attention

Every token forms three vectors: Query, Key, Value. Each Query is matched against every Key to produce attention weights.

Key insight

This is the only place tokens interact. Everything else is per-token.

06Stage

Multi-Head Attention

The model runs attention in parallel across 8 to 96 heads, each learning to attend to different relationships.

Key insight

Different heads learn different linguistic phenomena independently.

07Stage

Feed-Forward Network

After attention mixes information, each token passes through a small two-layer neural network independently.

Key insight

Attention routes information. The FFN is where knowledge resides.

08Stage

Residual & Norm

Wrapped around every sublayer are a residual skip connection and a layer normalization to keep activations stable.

Key insight

Skip connections let information bypass any single layer.

09Stage

Layer Stacking

Stack 12 to 120 transformer blocks. Each layer refines the representation, building abstraction on top of abstraction.

Key insight

Intelligence is the same mechanism, repeated, refining at every level.

10Stage

Output Projection

The final hidden state is multiplied by the unembedding matrix to produce a score (logit) for every token in the vocabulary.

Key insight

Every generation step asks 50,000 questions at once.

"42"
"love"
"simple"
"a"
11Stage

Softmax & Sampling

Logits become probabilities via softmax. Then the model picks one token. How it picks defines its 'personality'.

Key insight

The model never 'decides' — it samples. Randomness is a feature.

12Stage

Autoregression

The chosen token is appended to the input. The entire pipeline runs again, one token at a time, until a stop token appears.

Key insight

An LLM writes one token, then changes its mind about what comes next.
